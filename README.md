# Jesko Jets Clone (Awwwards-Inspired)

This project is a **non-commercial, educational clone** of Jesko Jets, an website originally featured on **Awwwards**.  
It was built to **practice advanced front-end development, animations, and interactive UI engineering**.

The goal of this project is to demonstrate **technical execution**, not to claim ownership of the original creative work.

---

## 🛠️ Tech Stack

- Next.js
- CSS
- GSAP
- Lenis

---

## ✨ Features

- Smooth interactive animations
- Scroll-based motion effects
- Responsive layout
- Performance-optimized assets
- Pixel-accurate UI recreation
- Modern front-end best practices

---

## 🎯 Purpose of This Project

This clone was built to:

- Improve **front-end engineering skills**
- Practice **complex animations & transitions**
- Study **award-winning UI/UX patterns**
- Showcase **portfolio-level craftsmanship**

This project is **not intended for commercial use**.
